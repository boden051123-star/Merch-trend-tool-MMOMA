
# Merch Trend Research Terminal - Hướng dẫn sử dụng

Ứng dụng này giúp bạn nghiên cứu Trend Merch by Amazon nhanh chóng trên 7 thị trường.

## Cách chạy ứng dụng:
1. **Lưu code:** Lưu file `index.html`, `index.tsx`, `constants.ts`, `App.tsx` và thư mục `components` vào một thư mục trên máy tính.
2. **Mở bằng Server:**
   - Cách dễ nhất: Dùng Extension **Live Server** trong VS Code.
   - Cách 2: Kéo thư mục vào [Netlify Drop](https://app.netlify.com/drop) để có ngay link web dùng mãi mãi.

## Tính năng chính:
- **7 Link Amazon tối ưu:** Tự động lọc sản phẩm mới nhất, hàng của Amazon (.com, .co.uk, .de, .fr, .es, .it + Top 100).
- **Copy Zip Code 1-Click:** Nhấn vào thẻ Zip code để tự động copy vào bộ nhớ đệm (clipboard).
- **Giao diện Pro:** Thiết kế theo phong cách Terminal chuyên nghiệp, tối ưu cho năng suất làm việc.
